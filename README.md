# 语音识别与蓝牙通信的灯控系统 python1201

## 项目概述

本项目是基于Python开发的一款集成了语音识别与蓝牙通信技术的灯控系统。系统通过识别语音文件中的指令，如“开灯”、“关灯”，并通过动画形式展示执行结果。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: MySQL
- **编译器**: PyCharm
- **Python版本**: 3.7

## 功能模块

- 语音识别指令解析
- 蓝牙通信控制灯具
- 动画效果展示

## 系统角色

- 用户：通过语音文件发送控制指令
- 系统：接收指令，控制灯具，并返回动画结果

## 运行环境

- Python 3.7
- Django
- MySQL数据库
- 支持蓝牙通信的环境

## 部署步骤

1. 安装Python 3.7版本
2. 安装Django框架
3. 配置MySQL数据库
4. 克隆或下载项目代码
5. 安装依赖库
6. 运行项目
